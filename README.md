## Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/Nagnie/InternAssignment_DevSamurai.git
cd InternAssignment_DevSamurai
```

### 2. Frontend setup
```
cd ../frontend
npm install
npm run dev
```
